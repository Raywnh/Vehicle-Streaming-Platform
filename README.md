# The Project  
 - Remote controlled vehicle mounted with a camera designed for video streaming 
 - Integrated with facial recognition features and a web application for vehicle control and rendering video footage.  
  
Built with:
 - React
 - ExpressJS
 - NodeJS
 - Flask
 - OpenCV
 - Apache
 - Raspberry Pi
